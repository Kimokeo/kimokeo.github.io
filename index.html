Crescendo

Designed to be a Github for musicians, Crescendo allows recording via MIDI devices, and allows you to download and upload your own MIDI files. Files can then be shared and utilized by other users.

Crescendo is made with Rails, JavaScript, Twitter-Bootstrap, jQuery, Underscore, Jazz-MIDI API and MySQL
